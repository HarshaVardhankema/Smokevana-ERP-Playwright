1. Introduction

Welcome to the ShipStation Integration Guide! This document will help you set up and use ShipStation's powerful shipping management features within your ERP system.

What is ShipStation?
ShipStation is a web-based shipping solution that helps you manage orders, print shipping labels, and track shipments from multiple carriers all in one place. Our integration allows you to seamlessly connect your ERP system with ShipStation's API.

Key Features Available:

🏢 Warehouse Management

Configure multiple warehouses with different shipping priorities and locations.

📦 Carrier Integration

Connect with multiple shipping carriers and configure their services.

💰 Rate Estimation

Get real-time shipping rates before creating shipments.

🏷️ Label Creation

Generate shipping labels directly from your sales orders.

✅ Address Validation

Automatically validate and clean shipping addresses.

❌ Shipment Voiding

Cancel shipments and void labels when needed.

2. Prerequisites

Before you begin, make sure you have the following:

1 ShipStation Account

You need an active ShipStation account. If you don't have one, sign up at shipstation.com

2 API Key (V2)

You'll need your ShipStation API Key (V2) to connect your ERP system. Here's how to get it:

  1. Log in to your ShipStation account
  2. Go to Settings → Account → API Settings
  3. Click on Generate API Key (if you don't have one)
  4. Copy your API Key - you'll need this in the next steps
3 Carrier Accounts

Make sure you have active accounts with shipping carriers (USPS, FedEx, UPS, etc.) connected in your ShipStation account.

⚠️ Important: Keep your API key secure and never share it publicly. The API key gives full access to your ShipStation account.

3. Getting Started

Follow these steps to access the ShipStation configuration in your ERP system:

1 Navigate to ShipStation Settings

In your ERP system, go to the main menu and look for "ShipStation" or "ShipStationAPI Configuration".

2 Access the Configuration Page

You should see a page titled "ShipStationAPI Configuration" with a table listing your warehouses (if any) and an "Add" button.

💡 Tip: If you don't see the ShipStation option in your menu, contact your system administrator. You need the "business_settings.access" permission to configure ShipStation.

4. Warehouse Configuration

Each warehouse represents a physical location from which you ship products. You can configure multiple warehouses with different priorities.

4.1 Adding a New Warehouse

1 Click the "Add" Button

On the ShipStation configuration page, click the "Add" button (usually in the top right corner).

2 Fill in Warehouse Details

A form will appear. Fill in the following information:

Field Description
Warehouse Select a warehouse name (Warehouse-1, Warehouse-2, or Warehouse-3)
Your Api-Key (V2) Paste your ShipStation API Key here
Priority Set priority number (higher numbers = higher priority). Used when multiple warehouses are available.
Usable Check this box to make the warehouse active and available for shipping
Contact Name Name of the contact person at this warehouse
Phone Contact phone number
Company Name Company name for this warehouse
Address 1 Street address of the warehouse
City/Locality City where the warehouse is located
State/Province State or province code (e.g., CA, NY, TX)
Postal Code ZIP or postal code
Country Code Two-letter country code (e.g., US, CA, GB)
3 Save the Warehouse

Click the "Save" button to create your warehouse configuration.

✅ Success! Your warehouse has been added. You should now see it in the warehouse list.

4.2 Verifying Your Warehouse Connection

1 Find Your Warehouse

In the warehouse list, locate the warehouse you just created.

2 Click "Verify"

Click the "Verify" button next to your warehouse. This will test the connection to ShipStation using your API key.

3 Check the Result

You should see a message indicating whether the connection was successful or failed.

  • Success: "Connection Test Successful" - Your API key is valid and working.
  • Failed: "Connection Test Failed" - Check your API key and try again.
⚠️ Note: If verification fails, double-check that:
  • Your API key is correct (no extra spaces)
  • Your ShipStation account is active
  • You have an internet connection

4.3 Editing a Warehouse

To modify an existing warehouse:

  1. Find the warehouse in the list
  2. Click the "Edit" button
  3. Update the fields you want to change
  4. Click "Save"

4.4 Deleting a Warehouse

To remove a warehouse:

  1. Find the warehouse in the list
  2. Click the "Delete" button
  3. Confirm the deletion
⚠️ Warning: Deleting a warehouse will remove all its configuration. Make sure you don't need it anymore before deleting.

5. Carrier & Services Setup

After creating a warehouse, you need to configure which carriers and shipping services are available from that warehouse.

5.1 Accessing Carrier Settings

1 Open Settings

In the warehouse list, click the "Settings" button (gear icon) next to the warehouse you want to configure.

2 View Available Carriers

A settings window will open showing all carriers available in your ShipStation account. This includes carriers like:

  • USPS (United States Postal Service)
  • FedEx
  • UPS
  • DHL
  • And other carriers you've connected to ShipStation

5.2 Selecting Carriers

1 Choose Carriers

Check the boxes next to the carriers you want to use from this warehouse. You can select multiple carriers.

2 Select Services

For each selected carrier, choose which shipping services you want to offer:

  • USPS: Priority Mail, First Class, Express Mail, etc.
  • FedEx: Ground, Express, 2Day, Overnight, etc.
  • UPS: Ground, 2nd Day Air, Next Day Air, etc.
3 Select Package Types

For each carrier, also select the package types you'll use:

  • Letter
  • Package
  • Flat Rate Envelope
  • Flat Rate Box
  • And other carrier-specific options
4 Save Configuration

Click "Save" to store your carrier and service selections.

💡 Tip: You can always come back and modify these settings later. It's a good idea to start with your most commonly used services and add more as needed.
✅ Configuration Complete! Your warehouse is now ready to use. The selected carriers and services will be available when creating shipments from this warehouse.

6. Using ShipStation Features

Now that your warehouse is configured, here's how to use ShipStation features in your daily operations:

6.1 Getting Shipping Rate Estimates

Before creating a shipment, you can get rate estimates to compare shipping costs:

1 Access Rate Estimation

When processing an order or creating a shipment, look for the rate estimation feature (usually in the shipping/packing section).

2 Enter Shipping Details

Provide the following information:

  • From Address: Your warehouse address (usually auto-filled)
  • To Address: Customer's shipping address
  • Package Weight: Total weight of the package
  • Package Dimensions: Length, width, and height
3 View Rates

The system will display available shipping rates from all configured carriers and services, allowing you to choose the best option.

6.2 Creating Shipping Labels

Once you've selected a shipping method, you can create a shipping label:

1 Select Shipping Method

From the rate estimates, choose the carrier and service you want to use.

2 Verify Address

The system can automatically validate and clean the shipping address to ensure it's correct.

3 Create Label

Click the button to create the shipping label. The system will:

  • Generate the label through ShipStation
  • Store the tracking information
  • Link the shipment to your order
4 Download Label

Download and print the shipping label. The label will be in PDF format, ready for printing.

6.3 Voiding a Shipment

If you need to cancel a shipment (e.g., customer canceled order, wrong address):

1 Find the Order

Locate the order with the shipment you want to void.

2 Void Shipment

Click the "Void Shipment" button (usually in the order details or shipping section).

3 Confirm

Confirm that you want to void the shipment. The system will:

  • Cancel the shipment with the carrier
  • Remove the shipping label
  • Update the order status
⚠️ Important: Not all shipments can be voided. Some carriers have time limits (usually within 24 hours) for voiding shipments. After that, you may need to process a return instead.

6.4 Viewing Shipment List

You can view all shipments created through ShipStation:

  • Navigate to the shipment list feature
  • View shipments from all active warehouses
  • See tracking information and shipment status

7. Troubleshooting

Here are common issues and their solutions:

7.1 Connection Issues

Problem: "Connection Test Failed" when verifying warehouse

Solutions:
  • Verify your API key is correct (no extra spaces before or after)
  • Check that your ShipStation account is active
  • Ensure you're using the V2 API key (not an old V1 key)
  • Check your internet connection
  • Try regenerating your API key in ShipStation

7.2 No Carriers Available

Problem: No carriers show up in the settings

Solutions:
  • Make sure you have carriers connected in your ShipStation account
  • Verify your carrier accounts are active in ShipStation
  • Check that your API key has the correct permissions
  • Try verifying the warehouse connection again

7.3 Rate Estimation Not Working

Problem: Can't get shipping rate estimates

Solutions:
  • Make sure the warehouse has carriers and services configured
  • Verify all address fields are filled correctly
  • Check that package weight and dimensions are valid
  • Ensure the warehouse is marked as "Usable"

7.4 Label Creation Fails

Problem: Can't create shipping labels

Solutions:
  • Verify all required address fields are complete
  • Check that the selected carrier and service are available
  • Ensure package dimensions and weight are within carrier limits
  • Verify your ShipStation account has sufficient credits/funds
  • Check that the warehouse API key is still valid

7.5 Can't Void Shipment

Problem: Shipment voiding fails

Solutions:
  • Check if the shipment is still within the void time window (usually 24 hours)
  • Some carriers don't allow voiding after the package is in transit
  • Verify the shipment ID is correct
  • Check that the warehouse API key is still valid

Getting Additional Help

If you continue to experience issues:

  1. Check the ShipStation status page: status.shipstation.com
  2. Review ShipStation's API documentation
  3. Contact your system administrator
  4. Reach out to ShipStation support if the issue is with their service

8. Best Practices

Follow these recommendations for optimal ShipStation usage:

8.1 Security

  • Protect Your API Key: Never share your API key or leave it visible in screenshots
  • Regular Updates: Regenerate your API key periodically for security
  • Access Control: Only give ShipStation configuration access to trusted users

8.2 Warehouse Management

  • Clear Naming: Use descriptive warehouse names (e.g., "Main Warehouse - New York")
  • Priority Settings: Set higher priorities for warehouses you use most frequently
  • Complete Information: Fill in all warehouse address fields for accurate rate calculations
  • Regular Verification: Periodically verify warehouse connections to ensure they're working

8.3 Carrier Configuration

  • Selective Services: Only enable services you actually use to avoid confusion
  • Package Types: Configure package types that match your actual packaging
  • Regular Review: Periodically review and update your carrier/service selections

8.4 Shipping Operations

  • Verify Addresses: Always use address validation before creating labels
  • Accurate Dimensions: Enter correct package dimensions for accurate rate calculations
  • Weight Accuracy: Use accurate weights to avoid carrier adjustments
  • Label Storage: Save shipping labels and tracking numbers for your records

8.5 Monitoring

  • Check Shipments: Regularly review your shipment list for any issues
  • Track Performance: Monitor which carriers and services you use most
  • Error Review: Address any shipping errors promptly
✅ Remember: Proper configuration and regular maintenance will ensure smooth shipping operations and help you save time and money!
